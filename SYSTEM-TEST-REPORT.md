### Pricing Logic Verification

**Quotation Pricing (quotations.service.ts):**
```typescript
const subtotal = items.reduce(
  (sum, item) => sum + quantity * unit_price, 0
);
const tax = items.reduce(
  (sum, item) => sum + quantity * unit_price * (tax_percent / 100), 0
);
const total = subtotal + tax;
```
✅ **Status:** Logic correct, tested with multiple scenarios

**Invoice Pricing (invoices.service.ts):**
```typescript
subtotal: quotation.subtotal,
tax: quotation.tax,
total: quotation.total,
```
✅ **Status:** Correctly inherits from quotation, no re-calculation errors
